需求分析:
	员工管理系统
功能分析:
	查询所有的员工信息
	根据编号查询员工信息
	修改员工信息(修改员工姓名)
	新增员工信息
	删除员工信息
数据库设计:
	员工信息表(emp)
Sql语句设计:
	查询所有的员工信息:
		select *from emp
	根据编号查询员工信息:
		select * from emp where empno=?
	修改员工信息(修改员工姓名):
		update emp set ename=? where empno =?
	新增员工信息:
		insert into emp values(?,?,?,?,?,?,?,?)
	删除员工信息:
		delete from emp where empno=?
功能开发:
	创建数据库操作功能类(不用的数据使用相同的数据库操作)
	创建程序入口类
